How do you know that this is legit?

I am trying to be as transparent as I can on website's Facebook page. Hover, you might still think that I am mining Bitcoin or something, but you can see the results of server's work on statistics page. But I can fake that data, right? Fortunately, folding@home provide us data for each donor with a custom username, here is our server. On this page you can find "Date of last Work Unit", so you can know if it's still active. If the last work unit was 1 day ago or even more recently, this means that the server is still active. You can also find the number of Work Units, so you can know how much processing power we use (here is everything you need to know about Work Units).
